Keyword/Concept Definition
Page Round Trip
DTD Document Type Definition files are used to validate XML documents, they describe the elements allowed in an XML document, and the order in which those elements must appear.
The <!DOCTYPE> declaration is used to reference an "external" DTD
Elements An indiviual component of an HTML document. Represents semantics or meaning.
Attributes Attributes are used to further configure an element
Ex) ID: a unique identifier, href: hyperlink for anchor tag
<a> The <a> tag defines a hyperlink, which is used to link from one page to another.
The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
<div> Often used as a container for other HTML elements.
Has no required attributes, but style, class and id are common.
<span> The <span> tag is used to group inline-elements in a document.
Block tag Define the structure of a page
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
Ex) <div>, <section>, <article>, <table>, <p>
Inline tag Used for formatting.
An inline element does not start on a new line and only takes up as much width as necessary.
Inline elements appear inside of a block-level element.
Ex) <a>, <time>, <span>, <code>
CSS Box Model
Inheritence Inheritance is associated with how the elements in the HTML markup inherit properties from their parent (containing) elements and pass them on to their children.
Grid 960 The 960 Grid System is an effort to streamline web development workflow by providing commonly used dimensions, based on a width of 960 pixels. There are two variants: 12 and 16 columns, which can be used separately or in tandem.
